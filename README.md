Mash up from multiple dotfiles.   


### Install my dotfiles:
```bash
curl -sL https://git.io/viqoU | bash
```

### Install my dotfiles + my favorites packages:
```bash
curl -sL https://git.io/viqoU | bash -s - --packages
```
